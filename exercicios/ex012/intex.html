Utilidades

  1. _blank = Vai abrir o link em uma nova janela em branco
  2. _self = vai abrir o link ou a janela no frame atual
  3. _top = vai desfazer todos os frames e abrir o destino no navegador completo
  4. _parent = Similar ao uso do top em uma referência à janela mãe
  5. Nome-do-frame caso esteja usando frames, indica o nome da janela a abrir

    6. como uso de frames está quase em desuso, normalmente só se utliza _blank ou _self

Mais Extras

  1. Next = indica que o link é para a próxima parte do documento atual
  2. Prev = Indica que o link é para a próxima parte do documento atual
  3. author = Indica que é um link para o site do autor do artigo atual
  4. external = Indica que é um link para outro site que não parte do site atual
  5. nofollow = Indica que é um link para um site não endossado, como um link pago
